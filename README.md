## Requirement
- composer


## Usage

- clone repository
> git clone https://github.com/mochijul/ProjectLivewire.git

- setup database & set env file
.env

- run migrate
> php artisan mingrate

- seed database
> php artisan db:seed

- run app
> php artisan serve
